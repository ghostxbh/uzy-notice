# uzy-notice
柚子云通知服务

## 架构
- JDK 1.8
- MongoDB 3.4
- SpringBoot 2.3.2
- swagger 3.0.0

- 前端
    + thymeleaf模版
    + layui
    + jQ
    + bootstrap

## 功能描述
- 测试接口数据流出
- 接口数据JSON格式化
- 流出数据通知
